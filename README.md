## Linux System Calls and Utilities

The goal of this project was to use and implement the main Linux System Calls and utilities such as: fork(2), pipe(2), openat(2), readdir(3), fdopendir(3), dirfd(3), opendir(3), xargs(1), execvp(3), wait(2).

This project was part of an assigment of the subject "Operating Systems", University of Buenos Aires.
